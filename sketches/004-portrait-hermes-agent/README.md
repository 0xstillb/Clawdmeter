## Variant: portrait-hermes-agent

### Design stance
Recast the Clawdmeter portrait screen in a Hermes Agent mood: darker tooling surfaces, sharper structure, bright yellow action emphasis, and a more operational docs-and-terminal feel.

### Key choices
- Layout: same 240×320 portrait structure and three states, but with a crisper top bar and more rectangular panel treatment
- Typography: Inter-heavy UI with monospace metadata and footer strings to echo Hermes docs + CLI culture
- Color: charcoal background, warm off-white text, yellow primary emphasis, green success signal, orange disconnect state
- Interaction: same three-state toggle with connected/disconnected header indicator mapped to usage vs pairing

### Trade-offs
- Strong at: feeling more like an AI agent / observability tool than a soft lifestyle dashboard
- Weak at: intentionally less firmware-faithful and less cozy than the locked BT-final design

### Best for
- Exploring a Clawdmeter direction that feels aligned with Hermes Agent branding and operational UX rather than Claude-like editorial warmth
