## Variant: portrait-faithful

### Design stance
Match the current firmware's small-screen layout as closely as possible, so the mockup acts like a visual checksum for the existing 240x320 port.

### Key choices
- Layout: two fixed-height stacked panels inside a 240×320 portrait frame
- Typography: large percentage first, small pill + muted reset line
- Color: current dark Anthropic-inspired palette from `theme.h`
- Interaction: toggle between live usage, pairing, and idle states

### Trade-offs
- Strong at: validating fit before hardware exists
- Weak at: not especially roomy; very close to the current compact layout

### Best for
- Checking whether the real current firmware proportions feel acceptable on the 2432S028 portrait screen
