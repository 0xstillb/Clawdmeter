## Variant: landscape-hermes-agent

### Design stance
Landscape 320×240 spin of the Hermes Agent portrait mockup. Uses the extra horizontal space to place Current and Weekly cards side-by-side for simultaneous visibility.

### Key choices
- Layout: 320×240 landscape with 2-column usage cards, slimmed header (40px), compact data-flow strip
- Typography: Inter-heavy UI with monospace metadata — scaled down slightly from portrait for landscape fit
- Color: same Hermes Agent palette — charcoal bg, blue #5a7aff primary, yellow #ffd53d secondary
- Animations: same pixel header animation, idle Z-particle system, data-flow stream, and card scan line
- Interaction: same 3-state toggle (usage / pair / idle)

### Trade-offs
- Strong at: showing both quotas at once, comfortable reset text, Hermes tooling feel
- Weak at: less vertical space for pairing/idle states — steps and hero are compacter

### Best for
- A rotated 2432S028-style display where quick side-by-side quota comparison matters more than tall poster-like layout
