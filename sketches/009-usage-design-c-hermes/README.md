## Variant: Design C Hermes

### Design stance
Keep Design C's two-up overview + lower matrix, but restyle it to feel like a Hermes agent console without changing the payload model.

### Key choices
- Layout: two equal summary cards for Current / Weekly, plus a lower telemetry frame
- Data model: still grounded in % + reset windows only
- Styling: Hermes blue primary, yellow weekly, scan-line + corner-frame details
- Hierarchy: top cards are the first read; lower panel is supporting telemetry

### Trade-offs
- Strong at: making Design C feel more branded and intentional
- Weak at: still duplicates percentages once in the lower telemetry frame for reinforcement

### Best for
- A more dashboard-like device screen where Current and Weekly should both read immediately, but the lower section still feels alive and technical
