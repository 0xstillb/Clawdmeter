## Variant: landscape-faithful

### Design stance
Use the extra horizontal space to put Current and Weekly side-by-side while keeping the same Clawdmeter semantics and assets.

### Key choices
- Layout: 320×240 landscape with two equal-width usage cards
- Typography: same family/tone as the portrait mockups, slightly tighter than portrait
- Color: same dark Anthropic-like palette and real extracted firmware assets
- Interaction: same 3-state toggle for usage / pairing / idle

### Trade-offs
- Strong at: keeping reset text comfortably inside the cards and making both quotas visible at once
- Weak at: pairing/idle states have less vertical room for longer copy

### Best for
- A rotated 2432S028-style device where quick side-by-side quota comparison matters more than a tall, poster-like layout
