## Variant: portrait-roomy

### Design stance
Keep the same portrait information architecture, but loosen spacing and make the subtext rhythm feel calmer and slightly more product-like.

### Key choices
- Layout: still two stacked cards in a 240×320 portrait frame
- Typography: slightly more breathing room around metrics and subtext
- Color: same dark palette with softer panel edges
- Interaction: same three-state toggle as the faithful version

### Trade-offs
- Strong at: easier visual scanning and a slightly less cramped feel
- Weak at: drifts a bit from the exact current LVGL geometry

- Comparing whether the 2432S028 should stay exactly as-ported or be given a small polish pass after first hardware bring-up
